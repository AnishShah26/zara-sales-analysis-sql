# Dataset

This project uses the **Zara Sales Dataset** from Kaggle.

Dataset link: https://www.kaggle.com/datasets/xontoloyo/data-penjualan-zara/data

## Files Included

- `zara.csv` — original semicolon-separated dataset downloaded from Kaggle.
- `zara_clean.csv` — cleaned comma-separated version with SQL-friendly column names.

## Main Fields

The dataset includes product-level information such as:

- product ID
- product position
- promotion flag
- product category
- seasonal flag
- sales volume
- product name and description
- price and currency
- section
